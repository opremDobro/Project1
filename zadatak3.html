Web čitači

Web čitač je aplikacija koja ne namenjena prikazu sadržaja ǌeb stranica.
Stranice se mogu dobaviti kako iy fajl-sistema, tako i putem TCP/IP mreze.
Protokol kojim komuniciraju web čitači i web server je HTTP protokol.Danas su najpoynatiji sledeči web čitači:

  1. Microsoft InternetExplorer(http://www.microsoft.com/ie)
  2. Mozila(http://www.mozilla.org)
  3. Opera(http://www.opera.com)

Obratite pažnju da su svi nazivi web čitača u gornjoj listi linkovi koji upućuju na
adrese navedene u zagradi.
